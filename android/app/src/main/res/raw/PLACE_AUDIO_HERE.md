# Audio file required

Place your custom sound file in this directory:

  android/app/src/main/res/raw/new_order.mp3

Rules:
- File name must be lowercase with no spaces (Android resource naming).
- Recommended format: .mp3 or .ogg (Android native; avoid .wav for Android).
- Any duration is fine — FLAG_INSISTENT repeats it automatically.
- Suggested sound: cash-register bell, kitchen bell, or loud chime.

Free sources: freesound.org, zapsplat.com (search "cash register" or "bell").
